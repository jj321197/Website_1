# 第九印記製棒｜網站圖片素材包

共 18 張 PNG 圖片，依網站用途分類。

## 資料夾

- `logo/`：透明背景橫式導覽列 Logo
- `hero/`：首頁 Hero 背景圖，文字與按鈕請使用 HTML 疊加
- `brand/`：品牌起源與工坊故事圖
- `products/`：六款直式商品圖
- `materials/`：楓木、白蠟木、樺木與複合材質特寫
- `craft/`：選材、車削、平衡、烙印四道工藝
- `cta/`：網頁結尾行動呼籲背景

## 建議使用方式

- Hero 與 CTA：`background-image` 搭配 `background-size: cover`
- 商品圖：Bootstrap Card 內使用 `img-fluid`
- 材質圖：建議固定 `aspect-ratio: 1 / 1` 與 `object-fit: cover`
- 工藝圖：建議固定 `aspect-ratio: 3 / 2` 與 `object-fit: cover`
- 所有標題、價格與按鈕使用 HTML，不要再寫入圖片
